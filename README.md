## Run It On Your Computer

### Project Setup With Maven

Projeyi klonlayın
```bash
  git clone https://github.com/Mduzgunn/AkbankCohort2.git
```

Proje dizinine gidin
```bash
  cd AkbankCohort2
```

Gerekli paketleri yükleyin
```bash
  mvn clean install
```

Projeyi çalıştırın
```bash
  mvn spring-boot:run
```
